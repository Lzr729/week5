# 阶段二：章节定位

## 1. 阶段目标

建立与公司设立、历史沿革、股本形成、整体变更、增资、股权转让和发行前股本结构有关的准确章节索引。

本阶段只确定标题、起止页、双页码映射、章节功能和证据锚点，不理解或标注具体资本事件，不提取投资机构、金额、股份数量、价格或持股比例。

## 2. 输入

- 源文件：`301563_云汉芯城_IPO招股说明书.pdf`
- 阶段一确认的页码规则：`PDF 阅读器页码 = 正文印刷页码 + 1`
- 阶段一确认的视觉风险：正文印刷第 52-70 页存在跨页图表和流程图。

## 3. 定位方法

1. 从目录确定二级章节入口。
2. 在正文中检索并人工确认各级标题首次出现页。
3. 用下一标题的位置确定上一标题内容覆盖的终止页。
4. 当上一章节尾部与下一章节标题出现在同一物理页时，允许两个章节的页码范围重叠。
5. 对“转下图”“续上图”等跨页结构，将所有连续页面纳入同一章节范围。
6. 每条记录同时保留 PDF 阅读器页码和正文印刷页码。
