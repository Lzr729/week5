# 阶段一：PDF 基础检查

## 1. 阶段目标

本阶段只确认招股说明书是否具备后续研究所需的基础可用性，不提取投资机构、投资金额、股份数量、交易价格、持股比例或最终 PE/VC 结果。

## 2. 输入

唯一主要事实来源：

```text
301563_云汉芯城_IPO招股说明书.pdf
```

## 3. 操作步骤

1. 打开 PDF，检查文件是否损坏及总页数。
2. 检查正文是否具有可选择、可复制、可搜索的文本层。
3. 定位目录首页和末页。
4. 通过目录、正文首页和目标章节锚点核对 PDF 阅读器页码与正文印刷页码。
5. 搜索与公司设立、历史沿革、股本形成、增资、股权转让、整体变更和发行前股本结构有关的关键词。
6. 人工查看正文印刷第 52-70 页的跨页图表、流程图和其他视觉结构。
7. 从目录筛选后续阶段的候选章节，但不解释具体资本事件。
