# 阶段四方法说明

## 1. 输入

本阶段使用以下输入：

- 阶段三固定的 CE-001 至 CE-014 候选事件；
- CE-014 下属 A、B、C、D、D+ 轮次级候选项；
- 《301563 云汉芯城招股说明书》对应章节、流程图、正文、表格及注释；
- 已通过阶段一和阶段二确认的 PDF 页码、章节定位及文本可检索性结论。

阶段三文件仅用于事件定位，事实判断以招股说明书原文为准。

## 2. 标注单位

每一条记录必须是能够独立说明其法律或股本效果的事件单位。

- 单一设立、增资、股权转让、资本公积转增或整体变更，通常作为一个事件；
- 同一候选节点包含不同法律行为时，可以保留父事件并拆分子事件；
- 协议签署、缴款、验资及工商登记若只是同一交易的履行节点，通常不拆成新的资本事件；
- 多名参与方或多笔明细本身不必然导致事件拆分；
- 特殊权利协议按轮次分别标注，但不将其误认为股本变动事件。

## 3. 时间标注规则

不同性质的时间不得混用，主要包括：

- 事件标题或流程图时间；
- 股东会、董事会或其他决议时间；
- 协议签署时间；
- 缴款、验资或审验时间；
- 工商登记或备案时间；
- 原文明示的实际完成时间；
- 审计、评估、确认函等辅助证据时间。

原文仅披露月份时，只保留月份精度，不推定具体日期。原文未披露的字段统一标注为“原文未披露”或“不适用”。

## 4. 参与方和变化方向

参与方按照其在具体事件中的角色登记，例如：

- 设立股东、出资人；
- 增资方、被增资公司；
- 转让方、受让方、标的公司；
- 发起人；
- 验资机构、登记机关；
- 特殊权利主体或协议相关方。

股东变化方向采用保守标注：

- `设立进入`：公司设立时成为初始股东；
- `新增`：此前未作为股东出现，本次进入；
- `退出`：有全部转让、持股归零或转让后股东表不再列示等证据；
- `持股增加`、`持股减少`：能够确认方向，但本阶段不计算最终比例；
- `不是股东变化`或`不适用`：机构、合同权利或程序性主体。

没有充分证据时，不将“持股减少”扩大解释为“退出”。

## 5. 证据规则

每项核心结论均应能够回溯到证据编号。证据记录至少包括：

- 事件编号；
- 章节编号和章节标题；
- PDF 阅读器页码；
- 正文印刷页码；
- 位置类型，如正文、表格、流程图或注释；
- 原文摘录或对表格的准确描述；
- 证据支持的字段；
- 证据限制和待复核事项。

跨页表格需保留完整范围，不能只引用其中一页。

## 6. 数值规则

本阶段只登记原文披露值，包括注册资本、增资额、股份数量、转让价款、持股比例、净资产和协议指标等。

- 保留原文单位；
- 保留数值对应主体和子事件；
- 不将“万元注册资本”“万元价款”“股”“%”混为同一口径；
- 不根据总额或比例反推未披露的参与方金额；
- 原文披露的价格或估值可以登记，但不重新计算；
- 需要勾稽和复算的事项留至阶段七。

## 7. 人工判断与原文事实分离

- `confirmed_facts`：原文可以直接确认的事实；
- `manual_judgment`：基于原文和前序已验收事件作出的事件边界或方向判断；
- `undisclosed_items`：招股说明书未披露的信息；
- `review_questions`：需要人工确认的关键判断。

任何推断不得伪装为原文披露。

## 8. 验收标准

单个事件通过验收需满足：

1. 已阅读全部相关原文；
2. 章节编号及双页码完整；
3. 核心事实可回溯到证据；
4. 各类时间分别处理；
5. 参与方角色有证据；
6. 复合事件拆分判断明确；
7. 股东变化方向标注保守且一致；
8. 原文值、人工判断和计算值严格分开；
9. 未提前开展阶段五至阶段八工作；
10. 不确定事项明确列示。

## 9. 本阶段输出

最终输出为一个关联式工作簿，而不是单张超宽表。事件主表、参与方表、证据表、数值表和复核清单通过 `event_id`、`evidence_id` 等字段建立关联，可作为后续时间线、股权结构快照、数值校验和 PE/VC 投资路径识别的结构化输入。
